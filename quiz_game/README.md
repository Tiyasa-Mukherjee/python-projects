# 🎮 Python Quiz Game

A command-line quiz game built with Python.

## Features
- Multiple choice questions (A/B/C/D)
- Score tracking with percentage
- Input validation — only accepts A, B, C or D
- Wrong answer review at end of quiz
- Congratulations message for perfect score

## How to Run

1. Clone this repository
   git clone https://github.com/Tiyasa-Mukherjee/python-projects.git

2. Navigate to quiz_game folder
   cd python-projects/quiz_game

3. Run the game 
   python quiz_game.py
