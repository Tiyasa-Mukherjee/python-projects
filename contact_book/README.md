# 📒 Contact Book

A command-line contact management system built with Python.

## Features
- Add new contacts
- Search contacts by name
- Delete existing contacts
- Update contact numbers
- Persistent menu until exit

## How to Run
1. Clone this repository
   git clone https://github.com/Tiyasa-Mukherjee/python-projects.git

2. Navigate to contact_book folder
   cd python-projects/contact_book

3. Run the program
   python contact_book.py
