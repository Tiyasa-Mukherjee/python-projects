# 🧮 Calculator

A command-line calculator with calculation history built with Python.

## Features
- Addition, Subtraction, Multiplication, Division, Modulus
- Calculation history — press H to view
- Division and modulus by zero handled gracefully
- Clean operator-based input (+ - * / %)

## How to Run
1. Clone this repository
   git clone https://github.com/Tiyasa-Mukherjee/python-projects.git

2. Navigate to calculator folder
   cd python-projects/calculator

3. Run the program
   python calculator.py

## Sample Output
1. Press + to perform addition
2. Press - to perform subtraction
3. Press * to perform multiplication
4. Press / to perform division
5. Press % to perform modulus
6. Press H/h to view history
7. Press X/x to exit

Enter your choice : +
Enter first number : 10
Enter second number : 5
10.0 + 5.0 = 15.0
