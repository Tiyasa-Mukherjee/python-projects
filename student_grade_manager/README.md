# 🎓 Student Grade Manager

A command-line student management system built with Python.

## Features
- Add students with their marks
- Calculate class average
- Display all students
- Show grades for each student
- Find the class topper
- Find students who failed
- Show student rankings

## Grading System
| Marks    | Grade |
|----------|-------|
| 90+      | A     |
| 80 - 89  | B     |
| 70 - 79  | C     |
| 60 - 69  | D     |
| 40 - 59  | E     |
| Below 40 | F     |

## How to Run
1. Clone this repository
   git clone https://github.com/Tiyasa-Mukherjee/python-projects.git

2. Navigate to student_grade_manager folder
   cd python-projects/student_grade_manager

3. Run the program
   python student_grade_manager.py
