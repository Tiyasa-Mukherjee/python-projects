# 📊 Word Frequency Counter

A command-line tool that counts word frequency in any text built with Python.

## Features
- Paste any text as input
- Counts frequency of every word
- Shows top 5 most common words
- Sorted by frequency in descending order

## How to Run
1. Clone this repository
   git clone https://github.com/Tiyasa-Mukherjee/python-projects.git

2. Navigate to word_frequency_counter folder
   cd python-projects/word_frequency_counter

3. Run the program
   python word_frequency_counter.py

## Sample Output
Enter your text : the quick brown fox jumps over the lazy dog the fox

Top 5 most common words are :
the : 3
fox : 2
quick : 1
brown : 1
jumps : 1

## Requirements
- Python 3.x
- No external libraries needed — pure Python!
